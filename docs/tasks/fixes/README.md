# Fixes — Correções e Bugs

Este diretório contém o rastreamento de **correções** e **bugs** do projeto.

Cada fix deve ter seu próprio subdiretório com o padrão `XXX-descricao-curta` (ex: `001-bug-login-mobile`), contendo:

- `TRACKER.md` — Status e checklist do fix
- Arquivos de tasks (opcional)

Consulte o [README principal](../README.md) para o guia completo de criação de fixes.
